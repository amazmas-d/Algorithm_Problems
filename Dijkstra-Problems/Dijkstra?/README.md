    For each edge (a, b, w):
        Add (b, w) to graph[a]
        Add (a, w) to graph[b]
    For i = 1 to n:

        pq.pop()

            CONTINUE

            IF dist[u] + w < dist[v]:
                dist[v] = dist[u] + w
                orig[v] = u
                pq.push((dist[v], v))

        RETURN -1
    ELSE:
        BUILD path from n to 1 using orig[]
        REVERSE path
        PRINT path
